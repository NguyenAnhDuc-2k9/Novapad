Program Guide

File:
New: Create a new document (Ctrl+N).
Open: Open text files, PDF, EPUB, MP3, and other supported formats (Ctrl+O).
Save: Save the document (Ctrl+S).
Save As: Save with a new name.
Save All: Save all documents (Ctrl+Shift+S).
Close Tab: Close the current document (Ctrl+W).
Recent Files: Open a recent file.
Reading: F5 Start reading, F4 Pause/Resume, F6 Stop reading.
Favorites during reading: F1 previous favorite voice, F2 next favorite voice.
Record Audiobook: Create an MP3 file from the current text (Ctrl+R).
Record Podcast: Record a podcast from microphone and/or system audio (Ctrl+Shift+R).

MP3 Audiobooks:
When you open an MP3 file, Novapad acts as an audio player.
Keyboard controls:
- SPACE: Play / Pause.
- RIGHT ARROW: Fast forward 1 minute.
- LEFT ARROW: Rewind 1 minute.
- UP ARROW: Increase volume.
- DOWN ARROW: Decrease volume.
- CTRL+B: Insert a bookmark at the exact time.

Edit:
Undo: Undo the last operation (Ctrl+Z).
Cut: Remove selected text and copy it to clipboard (Ctrl+X).
Copy: Copy selected text to clipboard (Ctrl+C).
Paste: Paste clipboard content into document (Ctrl+V).
Select All: Select all text (Ctrl+A).
Find: Search text in the document (Ctrl+F).
Find in Files: Search text across multiple files (Ctrl+Shift+F).
Find Next: Find next occurrence (F3).
Replace: Find and replace text (Ctrl+H).
Normalize Whitespace: Trim line edges and reduce blank runs (Ctrl+Shift+Enter).
Join Lines: Join lines in the paragraph or selection (Ctrl+Shift+J).
Hard Line Break: Reflow the paragraph or selection (Ctrl+Shift+H).
Strip Markdown Tags: Remove Markdown tags (Ctrl+Shift+M).
Order Items: Sort the paragraph or selection lines (Alt+Shift+O).
Keep Unique Items: Remove duplicate lines in the paragraph or selection (Alt+Shift+K).
Reverse Items: Reverse the lines in the paragraph or selection (Alt+Shift+Z).
Quote Lines: Add a prefix to lines in the paragraph or selection (Ctrl+Q).
Unquote Lines: Remove the prefix from lines in the paragraph or selection (Ctrl+Shift+Q).
Text Statistics: Shows characters, words, and lines for selection or document (Alt+Y).

View:
Show voices in editor: Show voice system, voice list, and multilingual filter.
Show favorite voices: Show the favorite voices list in the editor.
Text color: Change the text color (including light and dark colors).
Text size: Change the text size (small, normal, large, extra large, extra extra large).
When the list of entries is visible, pressing Tab moves the focus between the editor and the controls, while Shift+Tab moves it backward.
To add or remove an entry from favorites, use Tab to move to the list of entries and add or remove it using the Applications key or the right mouse button. From the list of favorite entries, you can also press the Applications key to remove entries that have already been added.
Keyboard shortcuts: it is possible to press F1 and F2 to switch between entries during reading, cycling through those added to favorites.

Insert:
Insert Bookmark: Save the current cursor position or audiobook time (Ctrl+B).
Manage Bookmarks: View, jump to, or delete saved bookmarks. When opening a file with bookmarks, the cursor or playback will automatically resume from the first available one.

Tools:
Batch Audiobooks: opens a window to convert a list of files (or entire folders) into audiobooks.
- Add files/folders to the queue.
- Choose destination folder and format (MP3 or WAV).
- Options to create subfolders and avoid overwrites.
- Press Start to process everything in the background.
Dictionary: Open the dictionary window. The list is navigated with Up/Down arrows.
Tab moves to the "Add entries to dictionary" button, activated with Enter or Space.
In the add/edit window: enter "Original word", then Tab to "Replacement word", then OK or Cancel (Enter/Space). ESC cancels.
In the list, Tab moves to "Edit entry" and "Remove selected entry".
Dictionary replacements are applied during reading and audiobook creation and are saved.
Import YouTube transcript: paste a URL, choose the language, and import the text into the editor (Ctrl+Y). Optional timestamps.
Accessible Terminal: opens a dedicated terminal with stable output for screen readers (Ctrl+Shift+T).
Shortcuts: Alt+I focuses input, Alt+O focuses output, Tab/Shift+Tab moves between input, output, and options.
Enter sends the input. Ctrl+C copies the output (or sends Ctrl+C to the program). Ctrl+L clears the output with confirmation.
Terminal options: auto-scroll, strip ANSI, announce new lines, beep after 1s idle, prevent sleep.
Program choices: Command Prompt (cmd.exe), PowerShell, Codex CLI (from Options).

RSS / Articles: opens the RSS/Articles window, add sources, expand a feed, and press Enter to import an article into the editor.
In the RSS window, select a feed or site and press Delete to remove it from the library.
From the context menu on a feed or site you can edit or delete an RSS or site.

Options:
Language: Change interface language.
Open Behavior: Open files in new tab or new window.
Voice: Choose voice for reading.
Multilingual Voices: Filter voice list.
Choose pitch, speed, and volume: Set pitch, speed, and volume for voices (both Microsoft and SAPI5). Applies to reading and audiobooks.
Move cursor during reading: Visually follow the text as it is being read.
Audiobook split by text: Split when the marker appears at the start of a line (case sensitive: "Chapter" and "chapter" are different). Example: with "Chapter" it creates one part per chapter, including author and introduction in the first part up to the first Chapter. If not found, no split.
Other options to split the audiobook: 2 parts, 4 parts, 6 parts, 8 parts.
